# Rendering the OG card

`web/og-card.png` is a committed 1200×630 screenshot of the design handoff's OG card
(`docs/design-handoff/standalone/og-card.html`). It is regenerated by hand whenever the
card design changes, not by CI.

Command used (run from the repo root; requires Playwright with its Chromium browser
installed, and network access for the card's Google Fonts):

```bash
npx playwright screenshot \
  --browser=chromium \
  --viewport-size=1200,630 \
  --wait-for-timeout=5000 \
  "file://$PWD/docs/design-handoff/standalone/og-card.html" \
  web/og-card.png
```

The 5 second wait gives the standalone page's bundler runtime and the web fonts
(JetBrains Mono, Space Grotesk) time to settle before capture. After rendering, verify:

```bash
file web/og-card.png   # must report exactly 1200 x 630
```

and eyeball the PNG against the standalone card in a browser (wordmark, headline,
sub line, diff pills top right, gradient bottom bar).
